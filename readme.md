## 说明

本仓库是KiCAD器件和封装库。

器件库的原件大部分都是从KiCAD的库复制而得，在库属性中添加了自己的条目，方便管理。

封装库大部分都是用官方的库，只有官方没有时，才在0WJQ.pretty中创建。

**本库只供参考，正确性各自验证。**

## KICad器件分类

对器件进行编码，以便量产管理，比如买料、贴片。

管理原则：功能不同、参数不同、外形不同、厂家不同、性能不同，单独编码。替代料？
